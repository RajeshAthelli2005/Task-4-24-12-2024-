VS(Visual Studio) Code

Introduction To VS Code :

Visual Studio Code, commonly referred to as VS Code, is an integrated development environment developed by Microsoft for Windows, Linux, macOS and web browsers. Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded version control with Git. Users can change the theme, keyboard shortcuts, preferences, and install extensions that add functionality.

Features Of VS Code :

Editing: Features include syntax highlighting, bracket-matching, auto-indentation, and multiple. cursors for simultaneous editing
Developer tooling: Includes IntelliSense code completion, debugging, semantic code understanding, and navigation
Version control: Built-in support for Git and other source control providers
Testing: Run automated tests and view test coverage
Customization: Users can change the theme, keyboard shortcuts, preferences, and install extensions
Split view: Allows users to work on two different files simultaneously or check the difference between two files.
Navigation bar: Also called breadcrumbs, this feature shows the file path and symbol path up to the cursor position
Plugins: Users can create their own plugins, such as Settings Sync to sync VS Code installations on different devices
Integrated command-line terminal: Users can open files, install extensions, change the display languages, and output diagnostics.

Shortcut keys

Shortcut keys Description
Ctrl+S
 Save the current file.
Ctrl+C
 Copy the selected text or item.
Ctrl+V
 Paste the copied content.
Ctrl+X
 Cut the selected text or item.
Ctrl+Y
 Redo the previously undone action.
Ctrl+z
 Undo the last action.
Ctrl+A
 Select all content in the current file.
Ctrl+F
 Open find box
